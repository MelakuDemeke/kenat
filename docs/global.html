ethiopianNumberConverter.js Converts Arabic numerals (natural numbers) to their equivalent Ethiopic numerals. Supports numbers from 1 up to 99999999. Example: toGeez(1); // '፩' toGeez(30); // '፴' toGeez(123); // '፻፳፫' toGeez(10000); // '፼'

Checks if the given Ethiopian year is a leap year. Ethiopian leap years occur every 4 years, when the year modulo 4 equals 3. This means years like 2011, 2015, 2019 (in Ethiopian calendar) are leap years.
