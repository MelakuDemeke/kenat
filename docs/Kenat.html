Class: Kenat

Kenat(ethiopianDateStropt)

Kenat - Ethiopian Calendar Date Wrapper A lightweight class to work with both Gregorian and Ethiopian calendars. It wraps JavaScript's built-in `Date` object and converts Gregorian dates to Ethiopian equivalents.

Constructor

new Kenat(ethiopianDateStropt)

Constructs a Kenat instance from an Ethiopian date string in 'yyyy/mm/dd' format, or defaults to the current Gregorian date converted to Ethiopian date if no argument is provided.
